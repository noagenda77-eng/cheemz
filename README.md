# Cheemz

A browser-based **Mauer Der Toten**-inspired zombie survival prototype built with Three.js.

## Current Features
- First-person movement with sprinting
- Wave-based zombie spawns
- Shooting, reloading, and ammo tracking
- Health, score, kill, and wave HUD
- Game over + restart flow

## Running Locally
Open `index.html` in a browser (or use a simple static server) and click to start.

## Controls
- **WASD**: Move
- **Mouse**: Aim
- **Click**: Shoot
- **R**: Reload
- **Shift**: Sprint
